Kuadru Legál-Operasionál Distribuisaun Kompeténsia ho Sistema Nasionál Integrado Monitorizasaun no Inspesaun

Modelu kooperasaun entre AIFAESA, Autoridade Munisipál sira (inklui RAEOA) no Ministériu Administrasaun Estatal — atu garante kobertura teritoriál boot liu, efisénsia iha fiskalizasaun, no governasaun bazeia ba dadus.

Fundasaun Legál no Kontextu Institusionál

 AIFAESA — Misaun Nasionál

AIFAESA garante, iha nível nasionál, kumprimu norma sira kona-ba seguransa alimentár, protesaun saúde públika, autentisidade produtu sira, no promosaun prátika ekonómiku neebé justu, inklui área metrolojia no padronizasaun.

 Servisu Munisipál — Atribuisaun Relevante

D.M. n.º 85/2023 prevê Servisu Munisipál Seguransa Alimentár no Servisu Munisipál Jestaun Merkadu no Turizmu, ho funsaun monitorizasaun, promosaun prátika di'ak, no fiskalizasaun lokál.

"Sobreposisaun parsial atribuisaun sira hatudu katak presiza koordinasaun ne'ebé estreitu liu no definisaun kompeténsia entre nível diferente sira klaru — esensiál hodi atu duplikasaun, konflitu institusionál no risku inefisiénsia, atu garante sistema fiskalizasaun ne'ebé koerente, efetivu no aliña ho objetivu nasionál sira."

Arkitetura Nasionál Inspesaun — Nível 3

1
Nível Munisipál
Baze Teritoriál — Liña Dahuluk

Servisu sira involve:

Seguransa Alimentár · Jestaun Merkadu no Turizmu · Protesaun Sivil · Ajénsia Munisipál Fiskalizasaun

  • Identifikasaun universál no rejistrasaun operadór lokál hotu-hotu
  • Verifikasaun preliminár: hijiéne, funsionamentu, lisensa sira
  • Monitorizasaun kontinua no presensa loroloron iha merkadu
  • Pré-klasifikasaun risku (Ki'ik / Médiu / Aas)
  • Edukasaun no kampaña prátika di'ak
  • Relatoriu obrigatóriu no estruturadu ba AIFAESA liu husi SNIMI
2
Servisus Deskonsentradus AIFAESA
Rejionál — Kluster 4

Zona/Rejiaun previstu:

RAEOA · Baukau (Vikeke, Lautém) · Kovalima (Ainaru, Bobonaru) · Díli alargadu (Aileu, Ermera, Manatuto, Likisá)

  • Validasaun no klasifikasaun definitiva risku (sistema pontuasaun)
  • Planeamentu semanal no fulan-fulan inspesaun bazeia ba prioridade
  • Ezekusaun inspesaun téknika formál ho koleksaun/rekolla prova/evidênsia
  • Resposta rápida (≤ loron 7) iha kazu risku aas
  • Koordinasaun no feedback kontinua ho munisípiu sira
3
AIFAESA Sentrál
Nasionál — Autoridade Másimu

Kompeténsia exkluziva:

  • Definisaun polítika, estratéjia no norma nasionál
  • Jestaun sistema nasionál risku (SNIMI)
  • Koordinasaun ho MAE no Autoridade Munisipál sira
  • Operasaun espesiál (fraude, rede ilegal sira)
  • Prosesu instruisaun no sansaun formál
  • Metrolojia no padronizasaun nasionál
  • Governasaun sistema dijitál nasionál

Seksaun 7

Mekanizmu Integridade no Antikorrupsaun

GPS Obrigatóriu
Submisaun hotu-hotu inklui koordenada GPS verifikável, lakohi permite inspesaun fiktísiu.
Foto Obrigatóriu
Evidénsia fotográfika mínima iha rejistrasaun ida-idak — ho timestamp automátiku integrado.
Timestamp Automátiku
Data no oras rejistadu automátikamente husi sistema — la bele edita husi utilizadór.
Blokeia Duplikadu
Sistema deteta no impedé submisaun duplikadu husi operadór hanesan iha períodu hanesan.
Distribuisaun Automátika
Atribuisaun aleatóriu no bazeia ba risku inspesaun sira — elimina seleksaun diskresianariu.
Auditoría Kruzada
Dadus munisipál no AIFAESA sira konfrontadu automátikamente hodi deteta anomalia.
Logs Imutável
Istória kompletu asaun iha sistema — imutável no asésivu ba auditoría esternu.
Alerta Komportamentu
Sistema deteta padraun suspeitu: ex. inspesaun barak laho evidénsia ka iha tempu rekorde.

Planu Implementasaun

I
Faze 1 · 0 – Fulan 3
Hahu Rápidu — Instrumentu Disponivel
  • Formuláriu dijitál ho Google Forms / KoboToolbox / ODK
  • Dashboard simplifikadu via Google Looker Studio / Metabase
  • Formasaun ajente munisipál no inspetór AIFAESA
  • Pilotu iha munisípiu 3 (Díli, Baukau, Kovalima)
  • Definisaun protokolu no checklist nasionál padronizadu
II
Faze 2 · Fulan 3 – 9
Sistema Web Dedikadu
  • Sistema web dedikadu (PHP + MySQL — stack ezistente Laragon/NAS)
  • Autentikasaun husi roles (munisipál / rejionál / sentrál / MAE)
  • Motor risku automatizadu integrado
  • Dashboard nasionál ho vista barak husi nível
  • Espansaun ba munisípiu hotu-hotu
  • Integrasaun ho baze SERVE no sistema lisensiamentu
III
Faze 3 · Fulan 9 – 18
Maturidade no Espansaun
  • App mobile nativu ho sinkronizasaun offline
  • GIS avansadu ho heatmap no analíza espasial
  • Integrasaun ho INSP-TL (laboratóriu analíza)
  • Portal denúnsia públiku online
  • Relatóriu automátiku ba órgaun soberania no parseiru internasionál
  • Integrasaun ho sistema saúde públika ba alerta epidemiolójiku
 Entregavel Téknika Disponivel ba Implementasaun Imediatu
  • Estrutura SQL kompletu sistema (schema baze dadus)
  • Kódigu PHP inisiál (formuláriu dijitál + painél administrativu)
  • Wireframe interface utilizadór (dashboard no formuláriu)
  • Prompt ba AI Agent jera sistema kompletu
  • Projetu Diploma Ministeriál ka Dekretu-Lei
  • Manuál operasionál ba inspetór no munisípiu sira

"Ho sistema ne'e, munisípiu ida-idak sai sentru ativu koleksaun dadus, enkuantu AIFAESA mantén kontrolu téknika no sansionatoriu. Ba dahuluk, nasaun sei iha visibilidade totál, iha tempu reál, kona-ba estadu konformidade ekonómiku, sanitáriu no alimentár iha territóriu tomak. Modelu ne'e la aumenta deit numeru inspesaun — transforma sistema fiskalizasaun husi modelu reativu no aleatóriu ba sistema intelijente, bazeia ba risku, integrado teritoriálmente no koerente legálmente, maximiza papel AIFAESA hanesan autoridade nasionál no uza munisípiu sira hanesan rede operasionál liña dahuluk."
